# 创建 GitHub 仓库步骤

## 快速步骤

1. 访问：https://github.com/new
2. 填写信息：
   - **Repository name**: `assets-management` (或你喜欢的名称)
   - **Description**: `资产管理系统 - 前端 + 后端`
   - **Visibility**: 选择 Public 或 Private
   - ⚠️ **不要勾选**：
     - ❌ Add a README file
     - ❌ Add .gitignore
     - ❌ Choose a license
3. 点击 **Create repository**

创建完成后，告诉我仓库名称，我会帮你配置并推送代码！

---

或者，如果你想使用 GitHub CLI 自动创建，运行：

```bash
gh auth login
```

然后我会帮你自动创建仓库。
