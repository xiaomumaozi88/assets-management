# 📍 获取 Supabase 连接字符串

## 🎯 当前位置

你现在在 **Database Settings** 页面。

## 📋 找到连接字符串的方法

### 方法 1：返回到项目主页

1. 点击顶部导航栏的 **"xiaomumaozi88's Project"**（在 "xiaomumaozi88's Org" 旁边）
2. 这会回到项目主页
3. 在项目主页，查找 **"Connect to your project"** 部分
4. 点击 **"API settings"** 按钮
5. 在设置页面中应该能看到连接字符串选项

### 方法 2：通过顶部导航

1. 在顶部导航栏，点击 **"Connect"**（右上角附近）
2. 这会打开连接向导或显示连接信息

### 方法 3：直接查找 Connection string

1. 在当前页面（Database Settings），向下滚动
2. 查找是否有 **"Connection string"** 或 **"Connection info"** 部分
3. 或者查找 **"Connection pooling"** 相关的连接信息

### 方法 4：通过 SQL Editor

1. 点击左侧导航栏的 **"SQL Editor"**（在 DATABASE MANAGEMENT 下）
2. 在 SQL Editor 页面，查找连接信息或设置图标

---

## ✅ 连接字符串应该在哪里

连接字符串通常显示在：
- 项目主页的 **"Connect to your project"** 部分
- **API settings** 页面
- 或者 **Database** → **Connection string** 部分

格式应该是：
```
postgresql://postgres.xxxxx:password@aws-0-<region>.pooler.supabase.com:5432/postgres
```
